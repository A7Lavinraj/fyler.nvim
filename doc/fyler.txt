================================================================================
INTRODUCTION                                                        *fyler.nvim*

Fyler.nvim is a file manager which can edit file system like a buffer.

How it different from |oil.nvim|?
- It provides tree view


================================================================================
CONFIGURATION                                                     *fyler.config*

Fyler supports plenty of options to customize. Following are default values

>lua

  ---@return string
  local function border()
    if vim.fn.has "nvim-0.11" == 1 and vim.o.winborder ~= "" then
      return vim.o.winborder
    end

    return "rounded"
  end

  local function winhighlight()
    return table.concat({

      "FloatBorder:FylerBorder",
      "FloatTitle:FylerBorder",

When `delete_to_trash` is enabled Fyler moves files and directories into the
operating system trash instead of permanently deleting them. Items that are
already inside the trash directory are removed permanently.
